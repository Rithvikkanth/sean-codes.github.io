About

Hi! I'm a high energy web software developer. I specialize in hand crafted vanilla JavaScript 2D Games and Apps. I started out writing software through game development. I spent every night for over 3 years self teaching myself how to use code to develop scalable network applications. From there my first game/project was an infinite scaling MMORPG that I wrote while livestreaming the development process. That project started from a 14 page development documentation I wrote and stuck to through the 50,000 lines of code the project reached. Since then I have worked on countless projects rangeing to higher end 2D game development to Android and IOS web applications for eCommerce. I live for the challenge of programming and will not stop until I master it to the best of my ability.
